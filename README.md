## About PandOS
PandOS is more than just a bot — it's a collaborative creation of the VPanda & BSOG teams. It automates server management, engages communities with fun commands, tracks Steam promotions, handles verification, and logs server activities. Designed to be scalable, reliable, and intuitive, PandOS ensures both admins and users have a smooth experience.

Developed in Python, the bot leverages discord.py, asyncio, requests, and BeautifulSoup. It uses lightweight JSON databases and AI algorithms to provide interactive chat and automation features.

## Key Features
- **Custom Verification:** Automatic verification and monitoring to maintain server security and quality.
- **Fun & Games:** Mini-games, leveling systems, and server economy to engage users.
- **Admin Tools:** Commands for moderation, warnings, announcements, and server management.
- **Steam Alerts:** Tracks Steam discounts and notifies users in channels.
- **Chat Interaction:** AI-powered random responses, message collection, and community enrichment.
- **Custom Media:** Image management with reactions and sharing options.
